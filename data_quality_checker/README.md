# Data Quality Checker

A simple Python script that validates CSV datasets before analysis.

## Features
- Detects missing values and duplicates
- Displays data types for each column
- Exports a structured JSON report
- Adds timestamp and summary info

## How to Run
1. Place your CSV file (e.g. `employees.csv`) in the same folder.
2. Run:
   ```bash
   python data_quality_checker.py
